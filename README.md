Road Accident Analysis Dashboard (Power BI)
Overview

This project presents an interactive Power BI dashboard analyzing road accident data across South Indian states. The dashboard is designed to identify patterns, highlight risk factors, and provide actionable insights to improve road safety.

Objective
Analyze accident trends by region, time, and severity
Identify high-risk areas and contributing factors
Support data-driven decision making
Improve overall road safety awareness
Key Features
State-wise accident distribution
Time-based analysis (day/night, peak hours)
Vehicle type impact on accidents
Severity classification (Low, Medium, High)
Fatality rate calculation
Interactive filters and slicers
Key Metric

Fatality Rate

Fatality Rate = (Fatal Accidents / Total Accidents) × 100
Dataset
Dataset: india_south_states_road_accident_dataset
Includes:
Total accidents
Fatal accidents
Vehicle types
Location details
Time of occurrence
Tools and Technologies
Power BI
Power Query (Data Cleaning)
DAX (Data Analysis Expressions)

Insights
Some states consistently report higher accident rates
Night-time accidents tend to be more severe
Heavy vehicles are often associated with higher fatality rates
Certain regions can be identified as accident hotspots
How to Use
Download the .pbix file
Open it using Power BI Desktop
Use filters and slicers to explore the data
Impact

This dashboard can support:

Government agencies in policy planning
Traffic authorities in identifying risk zones
Public awareness initiatives on road safety
